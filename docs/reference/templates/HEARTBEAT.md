# HEARTBEAT.md

Keep this file empty if you want heartbeat turns to do nothing.

Add only short, stable, recurring checks.

## Good Heartbeat Tasks

- check whether there are urgent reminders
- review inbox or notifications
- scan open work items
- perform lightweight workspace maintenance

## Avoid

- giant speculative planning
- noisy messages to the user without clear value
- repeating the same check too frequently

## No-Op Rule

If nothing needs attention, the correct outcome is silence or a minimal `HEARTBEAT_OK`-style acknowledgement.
